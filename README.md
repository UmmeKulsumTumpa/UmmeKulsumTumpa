## 👩‍💻 About Me

I'm an **Associate Software Engineer I at [Cefalo BD Ltd](https://www.cefalo.com/)**, a full stack web developer by practice and an AI/ML researcher by curiosity. I graduated in Software Engineering from the Institute of Information Technology, University of Dhaka, and now build production-grade web applications professionally while conducting research in Natural Language Processing.

At Cefalo, I've contributed to **2 in-house projects** and **2 client projects** — currently driving full stack development on a live client project. To broaden my impact further, I'm learning **Flutter** to contribute across mobile platforms as well.
